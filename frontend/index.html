IP Whitelist Test

Instructions:
1. Start your backend server (node server.js).
2. Serve this HTML file using a local web server (npx serve . or python -m http.server).
3. Open the served address in your browser (e.g., http://localhost:3000).
4. Click Register My IP to save your IP to the whitelist.
5. Click Fetch Whitelist to view all whitelisted IPs.
